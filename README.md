- **暗色主题**：保护眼睛，适合长时间查看

### 🔧 技术特点

- **Chart.js图表**：专业的数据可视化
- **Font Awesome图标**：丰富的图标系统
- **现代化CSS**：CSS Grid + Flexbox布局

## 🛠️ 部署说明

### GitHub Pages部署
1. 创建新的GitHub仓库：`money-dashboard`
2. 将本文件夹内容推送到仓库
3. 在仓库设置中启用GitHub Pages
4. 选择`main`分支作为源
5. 访问 `https://[你的用户名].github.io/money-dashboard`

### 本地运行
```bash
# 克隆仓库
git clone https://github.com/[你的用户名]/money-dashboard.git

# 进入目录
cd money-dashboard

# 使用任何HTTP服务器运行
# 使用Python
python3 -m http.server 8000

# 使用Node.js
npx serve .

# 使用PHP
php -S localhost:8000
```
